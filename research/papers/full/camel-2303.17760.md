# CAMEL: Communicative Agents for "Mind" Exploration of Large Language Model Society

Guohao Li, Hasan Abed Al Kader Hammoud, Hani Itani, Dmitrii Khizbullin, Bernard Ghanem

CAMEL (Communicative Agents for "Mind" Exploration of Large Language Model Society) is a novel communicative agent framework proposed to facilitate autonomous cooperation among communicative agents.

## Abstract

The rapid advancement of chat-based language models has led to remarkable progress in complex task-solving. However, their success heavily relies on human input to guide the conversation, which can be challenging and time-consuming. This paper explores the potential of building scalable techniques to facilitate autonomous cooperation among communicative agents, and provides insight into their "cognitive" processes. To address the challenges of achieving autonomous cooperation, we propose a novel communicative agent framework named role-playing. Our approach involves using inception prompting to guide chat agents toward task completion while maintaining consistency with human intentions. We showcase how role-playing can be used to generate conversational data for studying the behaviors and capabilities of a society of agents, providing a valuable resource for investigating conversational language models.

## Key Contributions

1. Introduces a novel communicative agent framework called role-playing
2. Offers a scalable approach for studying the cooperative behaviors and capabilities of multi-agent systems
3. Open-sources library to support research on communicative agents

## Methodology

The framework uses inception prompting to guide chat agents toward task completion while maintaining consistency with human intentions. The role-playing approach involves:

1. Defining roles for different agents
2. Using inception prompting to guide conversations
3. Facilitating autonomous cooperation among agents

## Applications

The framework can be used to generate conversational data for studying the behaviors and capabilities of a society of agents, providing a valuable resource for investigating conversational language models.
