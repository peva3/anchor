# Microsoft AI Agents for Beginners - AGENTS.md

Source: https://raw.githubusercontent.com/microsoft/ai-agents-for-beginners/main/AGENTS.md

## Overview

This repository contains an educational course teaching everything needed to build AI Agents with 15+ lessons covering fundamentals, design patterns, frameworks, and production deployment.

## Key Sections Found

### Project Overview
- Python 3.12+
- Jupyter Notebooks for interactive learning
- Microsoft Agent Framework (MAF)
- Azure AI Services: Microsoft Foundry, Azure AI Foundry Agent Service V2

### Setup Commands
- Virtual environment setup

- Running Jupyter Notebooks

- File organization per lesson

### Testing Instructions

### Common Gotchas
1. Python version mismatch (need 3.12+)
2. Environment variables (don't commit .env)
3. Package conflicts (use fresh venv)
4. Azure services region-specific features
